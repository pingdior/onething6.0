# OneThing 6.0 - AI智能目标管理助手

OneThing是一款面向创业者、自由职业者和学生群体的AI智能目标管理应用。通过将前沿AI技术与目标管理、心理学理论相结合，帮助用户实现高效的自我管理，及时发现认知偏差，调节异常情绪。

## 🎯 核心功能

### 1. 目标管理系统
- 对话式目标设定与分解
- 智能任务生成与调度
- 进度追踪与可视化
- 定期目标复盘

### 2. 情绪认知系统
- 实时情绪监测
- 智能干预建议
- 情绪日记
- 情绪-目标关联分析

### 3. AI伙伴系统
- 伙伴成长机制
- 每日互动仪式
- 深度对话系统
- 情感连接

## 📦 安装与运行

### 环境要求
- Node.js >= 16.0.0
- npm >= 8.0.0
- MongoDB >= 5.0

### 安装步骤

1. 克隆项目
```bash
git clone https://github.com/yourusername/onething6.0.git
cd onething6.0
```

2. 安装依赖
```bash
npm install
```

3. 配置环境变量
```bash
cp .env.example .env
# 编辑.env文件，填入必要的配置信息
```

4. 运行开发服务器
```bash
npm run dev
```

5. 构建生产版本
```bash
npm run build
```
